污染区生存 (SurvivalInPollution)
===
异星工厂的场景。在污染区会持续掉血。

特性 (features)
===
* 受到和污染值相关的伤害
* 打怪掉鱼，鱼换钱，钱买装备
* 自动吃鱼补血

模组/特性 (SoftMod/Features)
===
### from oarc
* regrowth_map
* autofill

### from ComfyFactorio
* antigrief
* group
* player_list
* poll
* score

### from mods
* autodeconstruct

计划(Futures)
===
* 场景介绍
* 集成 RSO
* 高污染值会刷出虫子
* 有提升抗性的装备（升级）
* 指示可能有矿的地方
* 污染颜色指示
* 写 IOC 和基于 global 存储的框架。
